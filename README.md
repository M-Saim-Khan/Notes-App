**Description:**
A full-stack Notes Application built with Django, Django REST Framework, and React. The application allows users to create an account, log in securely using JWT authentication, and manage their personal notes through a modern web interface.

**Features:**
- User registration and login
- JWT authentication
- Protected routes
- Create notes
- View notes
- Delete notes

**Made using:**
- Django
- Django REST Framework
- React
- SQLite

**Setup:**
**Backend:**
- pip install -r requirements.txt
- python manage.py migrate
- python manage.py runserver
**Frontend:**
- cd frontend
- npm install
- npm run dev
